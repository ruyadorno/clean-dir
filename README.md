> Command line util to clean a directory.


## Install

```sh
$ npm install --save clean-dir
```


## Usage

```js
var clean-dir = require('clean-dir');

clean-dir('Rainbow');
```

```sh
$ npm install --global clean-dir
$ clean-dir --help
```
